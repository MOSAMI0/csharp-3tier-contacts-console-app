# Contacts Management System (3-Tier Architecture)

This project is a C# console application that demonstrates the implementation of a 3-tier architecture:

- **Presentation Layer**: Console-based UI for testing and displaying data
- **Business Layer**: Handles application logic and validations
- **Data Access Layer**: Manages database operations using SQL Server

## Features
- Manage Contacts (Add, Update, Delete, Find)
- Manage Countries (Add, Update, Delete, Find by ID and Name)
- Existence checks (by ID and Name)
- Clean, formatted console tables with colors
- Proper handling of NULL values

## Technologies Used
- C#
- .NET
- SQL Server
- ADO.NET

## Purpose
This project was built to practice clean architecture principles and improve backend development skills using C# and database integration.
